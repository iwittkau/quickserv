Quickserv 
===

Quickly serve some files.


# Install

```
go get github.com/iwittkau/quickserv/cmd/quickserv
``` 

# Run

```
quickserv [-p= port number ] [-d= file path ]
``` 


## Example 

```
quickserv -p=8081 -d=/home/user/web
``` 

Run `quickserv` on port `8081` and serve folder `/home/user/web`
